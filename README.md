# 📚 Трекер прочитанных книг

Консольное приложение для учёта прочитанных книг с рейтингованием и статистикой.

## Возможности
- Добавление книг (автор, название, оценка 1-5, дата прочтения)
- Защита от дубликатов
- Просмотр всех книг
- Расчёт средней оценки
- Статистика по авторам
- Удаление книг
- Данные хранятся в JSON

## Установка и запуск
```bash
git clone <ваш-репозиторий>
cd <папка-проекта>
python main.py
